# RTK Query React TypeScript Starter

Vite + React + TypeScript + **Redux Toolkit** and **RTK Query**.

## Quickstart
```bash
pnpm i   # or npm i / yarn
pnpm dev
```

Open http://localhost:5173

## Configure API
Set `VITE_API_URL` in `.env` (see `.env.example`). Defaults to JSONPlaceholder for demo.

## What's inside
- Vite + React 18 + TS
- Redux Toolkit store (`src/store.ts`)
- RTK Query base API (`src/services/baseApi.ts`)
- Example Todos API slice (`src/services/todosApi.ts`)
- Counter slice example
- Strict TypeScript, ESLint, Prettier
